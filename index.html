angular-image-fallback

a directive that handle errors in images loading

Default beheviour.

Basic usage with default fallback.

With a custom fallback source.

How does it work?

It should be used with ng-src attribute only, because the value of src will be evaluted before the module and angular itself, has been loaded.

The directive will load the fallback image if the source is not specified or if the source produces an error, even if is dynamically changed.

You can use a custom fallback image passing to the attribute image-fallback, if the source you provided can't be loaded, the default one will be used.
